-Hay que añadir los jars que estan en la carpeta jars al build path.

-He usado sqlite para puntuaciones y archivo de base de datos hay que copiar en carpeta de proyecto (no en src
 pero en la carpeta donde sera la carpeta de src). El archivo esta en carpeta BD
 
-Como el proyecto esta creado en netbeans pueden ocurrir errores si no añadimos los jars a la hora de ejecutar en eclipse.

-Para importar a eclipse crear un proyecto en eclipse y copiar la carpeta Minesweeper que esta
 en src al src de proyecto.Despues importar jars y copiar el archivo que esta en BD (copiar en la carpeta de proyecto).

-javadoc esta en carpeta dist.Con archivo index.html que esta en dist/javadoc puedes ver la
 javadoc generada.
